----------------------------------------------------------------
Global Keys
----------------------------------------------------------------

F1	controls menu
F2	options menu

F5	enable/disable missile camera
F6	enable/disable rear camera
F7	show/hide the on screen hud
F8	save screenshot to ScreenShots folder

PAUSE	release the mouse pointer

SHIFT

	F12	switch between fullscreen and window mode

----------------------------------------------------------------
Multi Player
----------------------------------------------------------------

F3
F4

F9	send macro 1
F10	send macro 2
F11	send macro 3

SHIFT

	F9	edit macro 1
	F10	edit macro 2
	F11	edit macro 3

----------------------------------------------------------------
Singe Player
----------------------------------------------------------------

F3	load/save game menu
F4	save game menu

F9	show last text message
F10	show next text message
